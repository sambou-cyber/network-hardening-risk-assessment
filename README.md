# Network Hardening Risk Assessment

# Overview
This project presents a security risk assessment conducted for a social media organization following a major data breach caused by weak network security controls.

The assessment identifies key vulnerabilities related to authentication, firewall configuration, and access control, and recommends practical network hardening techniques to reduce the risk of future breaches.

## Skills Demonstrated
- Security risk assessment
- Network hardening techniques
- Authentication and access control analysis
- Firewall and network security fundamentals
- Preventive security planning

## Tools & Concepts
- Network hardening methods
- Firewall rule analysis
- Access control and MFA
- Security policy enforcement
